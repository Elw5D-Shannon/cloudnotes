说明如下：
getNamesAndTimes函数的功能是从某月的txt文件中统计当月所有参训飞行员的姓名与各自的飞行训练总时长，如对于图1所示txt文件，该函数返回值如图5所示，表示该月参加飞行训练的飞行员有：张三、李四四、王五、赵小六、孙七，他们这个月的飞行训练总时长分别为：271、185、277、319、289分钟

如图5所示，getNamesAndTimes函数有两个返回值：nameList和timeList，类型均为列表（list）、其元素类型分别为字符串（str）和整型（int）；nameList中存放了当月参训飞行员的姓名，timeList中存放了各飞行员当月的飞行训练总时长；其中，对飞行员姓名的顺序没有要求，但两个列表中的元素要有对应关系，例如，如果nameList的第0号元素为'张三'，则timeList的第0号元素存放的是“张三”当月的飞行训练总时长；

```python
#getNamesAndTimes函数的功能是从某月飞行数据txt文件中获取：
#1.当月参训飞行员姓名列表nameList
#2.各飞行员当月的飞行训练总时长列表timeList
#两个列表中的元素具有对应关系
def getNamesAndTimes(txtPath):
    nameList = [] #飞行员姓名列表
    timeList = [] #飞行训练总时长列表
    f = open(txtPath, 'r') #打开飞行数据txt文件
    ###########请在此行之后编写代码###########
    while True:
        line = f.readline()
        if line=='':
            break
        flds = line.split()
        name = flds[1]
        time = int(flds[2])
        if name not in nameList:
            nameList.append(name)
            timeList.append(time)
        else:
            idx = nameList.index(name)
            timeList[idx] = timeList[idx]+time
    ###########请在此行之前编写代码###########
    f.close() #关闭txt文件
    return nameList, timeList
#getTop3函数的功能是在getNamesAndTimes函数的基础上
#进一步获取当月飞行训练总时长最长的三名飞行员信息
#该函数的功能已经实现，不需完善、不要改动
def getTop3(txtPath):
    nameList, timeList = getNamesAndTimes(txtPath)
    tmpTimeList = list(timeList)
    tmpTimeList.sort()
    tmpTimeList.reverse()
    top3Names,top3Times=[],[]
    for i in range(3):
        idx = timeList.index(tmpTimeList[i])
        top3Names.append(nameList[idx])
        top3Times.append(timeList[idx])
    return top3Names,top3Times
    
if __name__ == '__main__':
```

补全函数**count(dirpath, keyword)**，该函数的功能是统计在路径为dirpath的文件夹中，文件名包含关键词keyword的文件总数量。具体说明如下：

1. 参数dirpath是一个字符串（str），用来表示某文件夹的相对路径，如dirpath为'sample/case1'时，对应的是当前文件夹下的sample文件夹下的case1文件夹（见下图）；参数keyword也是字符串（str），表示某一关键词，如'课件'；
2. 返回值num为整数（int），表示在dirpath对应文件夹下（包括子文件夹下），文件名包含关键词的文件数量，如dirpath为'sample/case1'、keyword为'课件'时，返回值num为7，因为case1文件夹下共有7个文件的文件名包含“课件”一词（见上图）；
3. 本题认为文件夹不是文件，count函数只统计满足条件的文件，不统计文件夹；
4. 不用考虑dirPath路径不存在的情况；
5. 在本题中，文件路径中的分隔符使用的是“/”；
6. 在程序文件中定义了getAllPaths(dirpath)函数，该函数的功能是获取dirpath对应文件夹下所有文件与文件夹的路径（具体说明见附1），在补全count函数时，可以调用getAllPaths函数，也可以不调用；

```python
import os
#不要改动getAllPaths函数
#函数功能：获取某文件夹下（其路径为dirpath）所有文件与文件夹的路径
def getAllPaths(dirpath):
    allPaths = []
    for root, dirs, files in os.walk(dirpath):
        for filename in files:
            path = os.path.join(root, filename)
            allPaths.append(path)
        for dirname in dirs:
            path = os.path.join(root, dirname)
            allPaths.append(path)
    return allPaths           
def count(dirpath, keyword):
    ###########请在此行之后编写代码###########
    num = 0
    allPaths = getAllPaths(dirpath)
    for path in allPaths:
        name = path.split('/')[-1]
        if os.path.isfile(path) and (keyword in name):
            num = num + 1
    return num
```

### os 模块常用属性和方法：

- os.sep： 可以取代操作系统特定的路径分隔符。windows下为 \ ；
- os.name： 字符串指示你正在使用的平台。比如对于 Windows，它是nt，而对于 Linux/Unix 用户，它是 posix；
- os.getcwd()： 函数得到当前工作目录，即当前 Python 脚本工作的目录路径；
- os.listdir(path)： 返回指定目录下的所有文件和目录名；
- os.remove(path)：该函数用来删除一个文件；
- os.linesep： 字符串给出当前平台使用的行终止符；
- os.path.split(path)： 该函数返回一个路径的目录名和文件名；
- os.path.isfile() 和os.path.isdir()函数分别检验给出的路径是一个文件还是目录；
- os.path.exists()： 函数用来检验给出的路径是否真地存在；
- os.curdir： 返回当前目录 (.)；
- os.mkdir(path)： 创建一个目录；
- os.makedirs(path)： 递归的创建目录；
- os.chdir(dirname)： 改变工作目录到dirname；
- os.path.getsize(name)： 获得文件大小，如果name是目录返回0L；
- os.path.abspath(name)： 获得绝对路径；
- os.path.normpath(path)： 规范path字符串形式；
- os.path.splitext()： 分离文件名与扩展名；
- os.path.join(path,name)： 连接目录与文件名或目录；
- os.path.dirname(path)： 返回文件路径；
- os.walk(top,topdown=True, 遍历迭代目录；
- os.rename(src, dst)： 重命名file或者directory src到dst；
- os.renames(old, new)： 递归重命名文件夹或者文件，像 rename() 。

### shutil模块常用属性和方法：

- shutil.copyfile( src, dst)： 从源src复制到dst中去。当然前提是目标地址是具备可写权限。抛出的异常信息为IOException. 如果当前的dst已存在的话就会被覆盖掉。
- shutil.move( src, dst)： 移动文件或重命名文件；
- shutil.copymode( src, dst)： 只是会复制其权限其他的东西是不会被复制的；
- shutil.copystat( src, dst)： 复制权限、最后访问时间、最后修改时间；
- shutil.copy( src, dst)： 复制一个文件到一个文件或一个目录；
- shutil.copy2( src, dst) ： 在copy的基础上再将文件的最后访问时间与修改时间也复制过来了；
- shutil.copy2( src, dst)： 如果两个位置的文件系统是一样的话相当于是rename操作，只是改名；如果是不在相同的文件系统的话就是做move操作；
- shutil.copytree( olddir, newdir, True/Flase) ： 把 olddir 拷贝一份 newdir ，如果第3个参数是True，则复制目录时将保持文件夹下的符号连接，如果第3个参数是False，则将在复制的目录下生成物理副本来替代符号连接；
- shutil.rmtree( src ) ： 递归删除一个目录以及目录内的所有内容。
