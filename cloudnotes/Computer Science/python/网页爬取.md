### 19年期末压轴

补全函数**getGradeByName(html, name)**，该函数的功能是对某次作业对应的HTML代码进行分析，提取某学员这次作业的最终成绩，具体说明如下：
1. 参数html为字符串（str），存储了某次作业页面对应的HTML代码，参数name为字符串（str），表示某学员的姓名，如'马闻昊'（不需考虑姓名不存在的情况）；
2. 返回值grade为字符串（str），表示姓名为name的学员的作业最终成绩，如参数html为图1对应的HTML代码、参数name为'马闻昊'时，grade的值为字符串'98.0'；
3. 当某学员未提交作业时，这次作业没有成绩，令返回值grade的值为None；
4. 若存在学员姓名相同的情况，则返回第1次出现时对应的最终成绩，例如，case1.txt中存在两个“王飞翔”，则返回第1个王飞翔的成绩'81.0'；
5. 枚举法不能得满分；
6. 答题过程中可能用到以下知识：
- find函数：实验教材第258页最后
- rfind函数：实验教材第259页小贴士
- findall函数：实验教材第262页第3段
- 正则表达式：实验教材第8.1.5节
```python
'''
  <!-- 成绩 -->
  <td class="hworkList70 c_red student_final_scor_info pr">
    95.0
        <div class="score-tip none tl" style="line-height: 18px; top: 10px;">
          <em></em>
          <font></font>
          作品最终评分为
          <span class="c_red">&nbsp;95.0&nbsp;</span>分<br>
          迟交扣分
          <span class="c_red">&nbsp;
            0
            &nbsp;</span>分<br>
          最终成绩为
          <span class="c_red">&nbsp;95.0&nbsp;</span>分<br>
        </div>
  </td>
'''
import re
def getGradeByName(html, name):
    ###########请在此行之后编写代码###########
    idx   = html.find('人已交')
    html  = html[idx:len(html)]
    idx   = html.find(name)
    html  = html[idx:len(html)]
    idx   = html.find('最终成绩为')
    html  = html[idx:len(html)]
    idx1  = html.find('&nbsp;')
    idx2  = html.find('&nbsp;</span>分<br>')
    grade = html[idx1+6:idx2]
    if grade == '':
        grade = None
    return grade 
    ###########请在此行之前编写代码###########
if __name__ == '__main__':
    path = input()
    name = input()
    f = open(path, 'r')
    html = f.read()
    f.close()
    grade = getGradeByName(html, name)
    print(name+':', grade)
```

### 22年期末压轴

补全函数**getNumByArea(html, area)**，该函数的功能是对图1所示网页的HTML代码进行分析，提取某来源国（地区）对应的确诊病例数。说明如下：
1. 参数html为字符串（str），存储了图1所示网页对应的HTML代码（如“sample/case1.txt”文件的所有内容）；参数area为字符串（str），表示某来源国（地区），如'英国'（不需考虑area不存在的情况）；
2. 返回值num为整数（int），表示来源国（地区）area对应的确诊病例数，如参数html为图1对应的HTML代码、参数area为'英国'时，num的值为79；
```python
import re #不一定要使用re库
def getNumByArea(html, area):
    ###########请在此行之后编写代码###########
    idx  = html.find(area)
    html = html[idx:len(html)]
    idx  = html.find('<span style="font-size:24px">')
    l    = len('<span style="font-size:24px">')
    html = html[idx+l:len(html)]
    idx  = html.find('<')
    num  = html[0:idx]
    #或num= re.findall(area+r'. ?<span style="font-size:24px">(. ?)</span>',html, re.S)[0]
    num  = int(num)
    return num #此句可根据需要修改或删除
    ###########请在此行之前编写代码###########
if __name__ == '__main__':
    path = input()
    area = input()
    f = open(path, 'r')
    html = f.read()
    f.close()
    num = getNumByArea(html, area)
    print(area+':', num)
```
