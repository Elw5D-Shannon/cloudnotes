### 数位运算
与，全为1才为1
或，全为0才为0
非，反过来
异或，相同为0，不同为1

码制
- 原码，正常二进制，最左边为符号位，正数为0，负数为1
  0的原码有两个 00···00 和10···00
  n位原码能表示$2^n-1$个整数
- 补码
  十位补码表示的数范围是从-512到511，共1024个不同的数。
  0的补码仅有00···00
  
- 反码
  正数的补码与反码和原码完全相同
  负数的反码是其原码的非，按位取反
  负数的补码是其反码+1


IP地址的A、B、C三类是为满足不同规模的网络需求而设计的，其区分的关键在于**网络号和主机号的长度划分**：
- **A类地址（对应大规模网络）**：
    - **特征**：第一个字节（8位）为**网络号**，且最高位固定为`0`。后续三个字节（24位）为**主机号**
    - **范围**：1.0.0.0 ~ 126.255.255.255（其中127.x.x.x为环回地址，用于本机测试）。 
- **B类地址（对应中等规模网络）**：
    - **特征**：前两个字节（16位）为**网络号**，最高两位固定为`10`。后续两个字节（16位）为**主机号**。
    - **范围**：128.0.0.0 ~ 191.255.255.255。
- **C类地址（对应小规模网络）**：
    - **特征**：前三个字节（24位）为**网络号**，最高三位固定为`110`。最后一个字节（8位）为**主机号**。
    - **范围**：192.0.0.0 ~ 223.255.255.255。


IP 地址用 32 位二进制表示。在使用时，IP 地址常采用点分十进制表示形式。
即把 32 位二进制分为 4 个字节，每个字节转换成一个十进制数，再用点号将四个十进制数连接起来，如下图所示。

32 位的 IP 地址由两部分组成:
- 前面若干位是网络号，用于标识设备位于的子网络
- 剩下的若干位是主机号，是设备在子网络中的编号。
**网络号和主机号均不能全为0或者全为1**

### 子网掩码
要从一个 IP 地址中提取网络号和主机号，需要用到子网掩码。
子网掩码决定了IP地址中哪些位是**网络部分**，哪些位是**主机部分**。
子网掩码也用 32 位二进制表示，且前面全部为 1，后面全部为 0，如 11111111 11111111 11000000 00000000，对应的点分十进制形式为 255.255.192.0。
假设现有点分十进制形式的 IP 地址和子网掩码，则利用子网掩码提取 IP 地址中的网络号和主机号过程如下：
　　1）将 IP 地址和子网掩码转换成 32 位二进制形式，分别用 a 和 b 表示；
　　2）令 c=a∧b，将 c 转换成点分十进制形式，即得网络号；
　　3）令 d=a∧¬b，将 d 转换成点分十进制形式，即得主机号。
　　例如，IP 地址为 168.16.127.253、子网掩码为 255.255.192.0，则计算过程如下：
　　1）IP 地址对应的二进制是 10101000 00010000 01111111 11111101，子网掩码对应的二进制是 11111111 11111111 11000000 00000000；
　　2）10101000 00010000 01111111 11111101 ∧ 11111111 11111111 11000000 00000000 = 10101000 00010000 01000000 00000000，对应的点分十进制形式是 168.16.64.0，这是网络号；
　　3）10101000 00010000 01111111 11111101 ∧ ¬11111111 11111111 11000000 00000000 = 00000000 00000000 00111111 11111101，对应的点分十进制形式是 0.0.63.253，这是主机号。
　　
在 Begin-End 区间实现parseIP(ip, mask)函数，说明如下：
　　1）参数 ip和mask是字符串，分别表示点分十进制形式的 IP 地址和子网掩码；
　　2）函数有两个返回值，依次是点分十进制形式的网络号和主机号；
　　3）允许使用 Python 提供的进制转换函数。
```python
########## Begin ##########
def DecToBin(d): #十进制到8位二进制，如168→10101000
    b = bin(d)
    b = b.replace('0b','')
    b = '0'*(8-len(b))+b #补满8位
    return b
    
def BinToDec(b): #8位二进制到十进制，如10101000→168
    return int(b, 2)
    
def BinIpToDottedIp(binIP):    #二进制到点分十进制，如10101000000100000111111111111101→168.16.127.253
    L = []
    for i in range(4):
        byte = binIP[8*i : 8*(i+1)] #取第i个字节
        byte = str(BinToDec(byte))  #将字节转换成十进制
        L.append(byte)
    return '.'.join(L)              #用点号连接4个部分
def DottedIpToBinIp(dottedIP):  #点分十进制到二进制，如168.16.127.253→10101000000100000111111111111101
    L = dottedIP.split('.')         #拆分成4个字节
    binIP = ''
    for byte in L:
        binIP = binIP+DecToBin(int(byte))
    return binIP
def AND(a, b): #逻辑与运算（第6关）
    c = ''
    for i in range(len(a)):
        if a[i]=='1' and b[i]=='1':
            c = c+'1'
        else:
            c = c+'0'
    return c
def NOT(a):  #逻辑非运算（第6关）
    c = ''
    for i in range(len(a)):
        if a[i]=='1':
            c = c+'0'
        else:
            c = c+'1'
    return c
def parseIP(ip, mask):
    ip = DottedIpToBinIp(ip)         #将点分十进制形式IP地址转换成二进制
    mask = DottedIpToBinIp(mask)     #将点分十进制形式子网掩码转换成二进制
    netID = AND(ip, mask)            #计算二进制形式的网络号
    netID = BinIpToDottedIp(netID)   #将二进制形式网络号转换为点分十进制形式
    hostID = AND(ip, NOT(mask))      #计算二进制形式的主机号
    hostID = BinIpToDottedIp(hostID) #将二进制形式主机号转换为点分十进制形式
    return netID,hostID
########## End ##########
ip = input()   #IP地址
mask = input() #子网掩码
netID, hostID = parseIP(ip, mask)
print('网络号：', netID)   #网络号
print('主机号：', hostID)  #主机号

```

### 划分子网
`/n` 表示**子网掩码的前 n 位为 1**。
`192.168.1.0/24` 表示前24位是网络位(前24位是1)（掩码 `255.255.255.0`）。

划分成更小子网时，比如 `/26`（掩码 `255.255.255.192`），就从主机位借了 2 位作为子网位。
假设一个网络 `192.168.1.0/24`（256个IP，254个可用主机）要划分为4个子网：
1. **确定需要多少子网**：4个子网 → 2^2=4，需要借2位主机位作为子网位。
2. **新掩码**：`24 + 2 = /26` → `255.255.255.192`。
3. **每个子网大小**：主机位还剩 32−26=632−26=6 位 → 26=6426=64个IP。
4. **子网划分结果**：
    - 子网1：`192.168.1.0/26`（范围 0~63，网络地址 `.0`，广播地址 `.63`）
    - 子网2：`192.168.1.64/26`（范围 64~127，网络地址 `.64`，广播地址 `.127`）
    - 子网3：`192.168.1.128/26`
    - 子网4：`192.168.1.192/26`

>[!example]
>现有一台主机的IPv4地址为170.108.200.24，子网掩码为255.255.240.0，则其网络号为（ ）。（此处说的网络号包括子网号）

解析：
首先，**将IPv4地址和子网掩码转换为二进制形式**：
    **170.108.200.24 的二进制表示为：10101010.01101100.11001000.00011000**
    **255.255.240.0 的二进制表示为：11111111.11111111.11110000.00000000**

接下来，将IPv4地址和子网掩码进行**按位与运算**：
    10101010.01101100.11001000.00011000  & 
    11111111.11111111.11110000.00000000
    10101010.01101100.11000000.00000000
**将结果转换回十进制：**
- 10101010 -> 170
- 01101100 -> 108
- 11000000 -> 192
- 00000000 -> 0
所以，**网络号为：170.108.192.0**
