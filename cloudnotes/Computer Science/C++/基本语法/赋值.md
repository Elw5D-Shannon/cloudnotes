### 直接赋值 `=`

- **只能**用于：
    - 另一个同类型容器
    - 初始化列表（C++11起）
- 语法更简洁直观

### `assign()` 函数

- **更多用途**：
    - 初始化列表
    - 指定数量和值
    - 迭代器范围
    - 更灵活的参数形式

### vector:
```cpp
#include <iostream>
#include <vector>
#include <list>
using namespace std;
int main() {
    vector<int> v;
    
    // ========== 直接赋值 ==========
    v = {1, 2, 3, 4, 5};          // ✅ 初始化列表
    
    vector<int> v2 = {10, 20, 30};
    v = v2;                         // ✅ 另一个vector
    
    // v = 5, 10;                    // ❌ 错误！不能这样
    // v = v2.begin(), v2.end();     // ❌ 错误！不能直接赋迭代器
    
    // ========== assign() ==========
    v.assign({1, 2, 3, 4, 5});     // ✅ 初始化列表
    
    v.assign(5, 100);               // ✅ 5个100：{100,100,100,100,100}
    
    v.assign(v2.begin(), v2.end()); // ✅ 迭代器范围
    
    list<int> lst = {1, 2, 3};
    v.assign(lst.begin(), lst.end()); // ✅ 可以从不同类型的容器赋值
    
    // assign(10)                    // ❌ 错误！至少需要两个参数
}
```

### string:
```cpp
#include <iostream>
#include <string>
using namespace std;
int main() {
    string s;
    
    // 直接赋值
    s = "Hello";                    // ✅ C字符串
    s = string("World");            // ✅ string对象
    s = {'H', 'i'};                  // ✅ 初始化列表（C++11）
    
    // assign() - 更多功能
    s.assign("Hello");               // ✅ C字符串
    s.assign("Hello", 3);            // ✅ 前3个字符："Hel"
    
    s.assign(5, 'A');                // ✅ 5个A："AAAAA"
    
    string t = "World";
    s.assign(t);                     // ✅ string对象
    s.assign(t, 1, 3);               // ✅ 子串："orl"
    
    s.assign({'H', 'e', 'y'});        // ✅ 初始化列表
    
    return 0;
}
```


### 场景对比
```cpp
// 场景1：简单赋值，用 =
vector<int> v;
v = {1, 2, 3, 4, 5};  // 简洁明了

// 场景2：需要重复值，用 assign()
v.assign(10, 42);  // 10个42，一行搞定

// 场景3：从其他容器赋值，用 assign()
list<double> ld = {1.1, 2.2, 3.3};
vector<double> vd;
vd.assign(ld.begin(), ld.end());  // 不同类型容器

// 场景4：string子串，用 assign()
string s;
s.assign("Hello World", 5);  // "Hello"

// 场景5：已有同类型容器，用 =
vector<int> v1 = {1, 2, 3};
vector<int> v2;
v2 = v1;  // 简单直接
```

|特性|直接赋值 `=`|`assign()` 函数|
|---|---|---|
|**语法简洁性**|✅ 更简洁|⚠️ 稍显冗长|
|**同类型容器**|✅ 支持|✅ 支持|
|**初始化列表**|✅ 支持（C++11）|✅ 支持（C++11）|
|**指定数量+值**|❌ 不支持|✅ 支持|
|**迭代器范围**|❌ 不支持|✅ 支持|
|**不同容器类型**|❌ 不支持|✅ 支持|
|**字符串子串**|❌ 不支持|✅ 支持|
