
### **一、使用 `iostream` 库中的 `std::cout`** 

std::cout定义在`<iostream>`头文件中，支持多种数据类型的输出。
std::cout本身并不直接支持复杂的格式化，但可以通过`<iomanip>`头文件中的操纵符来实现格式化。
iomanip是**Input/Output Manipulators的缩写（输入/输出操纵符）。**

#### 设置输出宽度 `std::setw`(int n)

`std::setw` 用于设置下一个输出项的宽度（字符数）。如果输出的内容小于指定宽度，默认会用空格填充，如下所示：

```cpp
#include <iostream>
#include <iomanip>  // 包含 std::setw

int main() {
    int num = 10;
    std::cout << std::setw(10) << num << std::endl;  // 宽度为8，右对齐
    return 0;
}
```

输出结果：      10。
输出结果总宽度为8，结果"10"前面有6个空格。

#### 设置填充字符`std::setfill`(char c)

用于设置填充字符，通常与`std::setw`一起使用，如下所示：

```cpp
#include <iostream>
#include <iomanip>

int main() {
    int num = 10;
    std::cout << std::setw(8) << std::setfill('*') << num << std::endl;
    return 0;
}
```

输出结果为：`******10`。

#### 设置浮点数精度 `std::setprecision`(int n) 

std::setprecision用于设置浮点数的输出精度，输出有效数字位数。

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.141592653589793;
    std::cout << std::setprecision(4) << pi << std::endl;  
    return 0;
}
```

输出结果为 3.142。

这里需要特点注意的是，这里输出的是有效数字的位数，即有多少个数字，如果要输出小数点后几位，要加上fixed。如下面代码所示：

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.141592653589793;
    std::cout<< std::fixed << std::setprecision(4) << pi << std::endl;  
    return 0;
}
```

输出：3.1416

#### 对齐方式

默认情况下，`std::cout` 是右对齐的。可以通过 `std::left`(左对齐，输出内容在左)和 `std::right`（右对齐，输出内容在）控制对齐方式（注：目前还没有居中对齐）。
如下面代码所示：

```cpp
#include <iostream>
#include <iomanip>
 
int main() {
 int num = 42;
 std::cout << std::setw(10) <<std::setfill('*')<< std::left << num << std::endl; // 左对齐
 std::cout << std::setw(10) <<std::setfill('*')<< std::right << num << std::endl; // 右对齐
 return 0;
}
```

输出 ：
`42********  //左对齐`

 `********42  //右对齐`

#### 固定小数点和科学计数法

`fixed` 和 `scientific` 可以控制浮点数的输出格式。

```cpp
#include <iostream>  
#include <iomanip>  
  
int main() {  
    double num = 123456789.0;  
    std::cout << "Fixed: " << std::fixed << num << std::endl;  
    std::cout << "Scientific: " << std::scientific << num << std::endl;  
    return 0;  
}  
```

**输出结果:**

Fixed: 123456789.000000
Scientific: 1.23456789e+08


####  进制转换

可通过 `std::dec`、`std::hex` 和 `std::oct` 分别以十进制、十六进制和八进制输出整数

```cpp
#include <iostream>
int main() {
 int num = 42;
 std::cout << std::dec << num << std::endl; // 十进制，输出 42
 std::cout << std::hex << num << std::endl; // 十六进制，输出 2a
 std::cout << std::oct << num << std::endl; // 八进制，输出 52
 return 0;
}
```

#### 科学计数法

使用 `std::scientific` 可以将浮点数以科学计数法输出，如下面代码所示：

```text

int main() {
 double num = 123456.789;
 std::cout <<std::setprecision(2)<< std::scientific << num << std::endl; 
 return 0;
}
```

 以上 输出 1.23e+05 

#### 输出布尔值 (std::boolalpha)

如果想输出true，可以使用std:: std::boolalpha。

### **二、使用 printf**

printf是 C 语言中用于格式化输出的函数，属于标准库 `<stdio.h>` 。
格式如下：
`printf (<格式控制串>,  <参数列表>) ;`

 <格式控制串>可以包含3种类型的字符：格式指示符、转义字符、普通字符。其中：
   - 格式指示符的作用是将参数列表中相应的输出数据转换为指定的格式输出
   `% <标志> <域宽> <精度> <转换说明符>`
   - 转义字符按照其含义输出相应的特殊符号
   - 普通字符则按照原样输出
<参数列表> 存放输出数据列表，有多个输出数据时中间用`,`分隔。格式指示符的数量、顺序与输出数据相对应。
    

printf 函数执行时，根据格式控制串中的格式指示符、转义字符和普通字符等，从左到右依次进行处理：遇到普通字符，则按照原样输出；遇到转义字符，则按照其含义输出相应的符号；遇到格式指示符，则将对应的输出数据按照格式指示符指定的方式输出。

格式指示符的一般形式为： `% <标志> <域宽> <精度> <转换说明符>`

其中，`<标志>`、`<域宽>`和`<精度>`都是可选项，可以不出现。


%[标志][宽度][.精度][长度]类型

- **类型（必填）**

| 符号 | 作用        | 示例                    | 输出       |
| -- | --------- | --------------------- | -------- |
| %d | 整数        | printf("%d", 42);     | 42       |
| %f | 浮点数（默认6位） | printf("%f", 3.14);   | 3.140000 |
| %s | 字符串       | printf("%s", "Hi");   | Hi       |
| %c | 字符        | printf("%c", 'A');    | A        |
| %x | 十六进制小写    | printf("%x", 255);    | ff       |
| %p | 指针地址      | printf("%p", &amp;n); | 0x7ff... |
| %% | 输出 %      | printf("%%");         | %        |

- **常用标志**

· `-`：左对齐（默认右对齐）
`printf("%-5d", 42);` → `42`

· `0`：用0填充宽度
`printf("%05d", 42);` → `00042`

· `+`：显示正负号
`printf("%+d", 42);` → `+42`

· `.精度`：控制小数或字符串长度
`printf("%.2f", 3.1415);` → `3.14`
`printf("%.3s", "Hello");` → `Hel` 

- **宽度与对齐**

· `%5d`：至少占5个字符宽度，不足补空格
`printf("%5d", 42);` → `42`

· `%-5d`：左对齐，占5字符
`printf("%-5d", 42);` → `42` 

- **长度修饰符**

· `%ld`：长整型 → `long num = 100L;`

· `%lld`：长长整型 → `long long num = 100LL;`

- **特殊标志**

| 标志  | 说明                                                   | 示例代码               | 输出   |
| --- | ---------------------------------------------------- | ------------------ | ---- |
| #   | 特殊格式：<br>  - %o 前导0<br>  - %x 前导0x<br>  - %f 强制显示小数点 | printf("%#x", 42); | 0x2a |

- **综合示例**

```cpp
#include <stdio.h>
 
int main() {
    int num = 42;
    float pi = 3.14159;
    char str[] = "Hello";
 
    printf("[%+-10d]\n", num);    // 左对齐+符号，占10位 → [+42      ]
    printf("[%#010x]\n", num);    // 十六进制带前导0x，占10位 → [0x0000002a]
    printf("[%10.3s]\n", str);    // 字符串取3字符，占10位 → [       Hel]
    printf("[%.2f%%]\n", pi*100); // 保留两位小数 → [314.16%]
    return 0;
}
```


此外还有输出至文件流的`fprintf`可供使用

```cpp
# include <stdio.h>

int main() {

    int value = 42;  
    printf("Value: %d\n", value);
    // 输出到标准输出  
    
    int age = 25;
    
    // 输出到标准输出（屏幕）- 等同于 printf
    fprintf(stdout, "年龄：%d\n", age);
    
    // 输出到标准错误（屏幕，但通常用于错误信息）
    fprintf(stderr, "错误：年龄不能为负数！\n");
    
    // 输出到文件
    FILE *fp = fopen("output.txt", "w");
    if (fp != NULL) {
        fprintf(fp, "年龄：%d\n", age);
        fprintf(fp, "这是一行写入文件的内容\n");
        fclose(fp);
    }
    
    return 0;
}
```
