> **摘要：C++ 循环控制语句是实现代码重复执行的关键结构，可以根据特定条件多次执行同一段代码。C++主要提供三种类型的循环语句：`for` 循环、`while` 循环和 `do-while` 循环，以及两个用来控制循环流程的关键字 `break` 和 `continue`。

# 一、for 循环语句

`for` 循环是一种结构化的控制语句，用在已知循环次数或在特定范围内迭代操作的场景。

## 1.1 for 循环的基本语法与工作原理

`for` 循环的基本语法结构如下：

```cpp
for (初始化表达式; 条件表达式; 更新表达式) {
    // 待执行的代码块（循环体）
}
```

- **初始化表达式**：在循环开始前执行一次，用来声明并初始化循环控制变量。
- **条件表达式**：在每次循环迭代开始前进行判断。如果条件为 `true`，则执行循环体；如果为 `false`，则终止循环。
- **更新表达式**：在每次循环体执行完毕后执行，更新循环控制变量的值。

**`for`** **循环的执行流程：**

1. 执行 **初始化表达式** (仅一次)。
2. 评估 **条件表达式**。如果条件为 `true`：执行 **循环体** 中的代码；执行 **更新表达式；**返回步骤 2。如果条件为 `false`：终止循环，程序继续执行 `for` 循环后的语句。

**示例：使用** **`for`** **循环输出数字 1 到 5**

```cpp
#include <iostream>

int main() 
{
    // 循环从 i=1 开始，当 i 小于等于 5 时继续，每次循环后 i 递增 1
    for (int i = 1; i <= 5; i++) {
        std::cout << i << std::endl;
    }
    return 0;
}
```

## 1.2 break 和 continue 关键字在 for 循环的应用

`break` 和 `continue` 是循环控制关键字，可以在循环体内改变正常执行流程。

**`break`** **关键字：** 立即终止当前所在的循环（`for`、`while`、`do-while` 或 `switch` 语句），程序控制流跳转到循环体后的第一条语句。

**示例：使用** **`break`** **在达到特定条件时提前退出循环。**

```cpp
#include <iostream>

int main() {
    int sum = 0;
    // 计算 0 到 100 的和，但在 i 等于 50 停止
    for (int i = 0; i <= 100; i++) {
        if (i == 50) {
            std::cout << " i = 50，退出循环。" << std::endl;
            break; // 跳出 for 循环
        }
        sum += i;
    }
    std::cout << "累加的和 (只加到 49): " << sum << std::endl;
    return 0;
}
```

**`continue`** **关键字：** 跳过当前循环迭代，直接进入下一次循环迭代。

**示例：用** **`continue`** **跳过指定迭代。**

```cpp
#include <iostream>

int main() {
    int sum = 0;
    for (int i = 0; i <= 100; i++) {
        if (i == 50) {
            std::cout << "跳过 i = 50" << std::endl;
            continue;
        }
        sum += i;
    }
    std::cout << "累加的和 (跳过 50): " << sum << std::endl;
    return 0;
}
```




## 1.3 for 循环的嵌套

C++的`for` 循环可以嵌套，一个 `for` 循环可以包含另一个 `for` 循环。

**示例：用嵌套** **`for`** **循环打印 1 到 10 的乘法表**

```cpp
#include <iostream>

int main() 
{
    for (int i = 1; i <= 10; ++i) {
        for (int j = 1; j <= 10; ++j) {
            std::cout << i * j << "\t";
        }
        std::cout << std::endl;
    }
    return 0;
}
```

**注意：**

- 嵌套循环会增加代码的执行次数。
- 嵌套循环要特别注意循环变量的命名。
- 避免死循环。


## 1.4 for 循环的应用

### 1.4.1 逐行读取字符串

```cpp
#include <iostream>
#include <string>
#include <vector>
int main() {
    int n;
    std::cin >> n;  // 读取第一行的正整数
    
    std::vector<std::string> strings(n);  // 预先分配空间
    
    // 读取 n 行字符串
    for (int i = 0; i < n; i++) {
        std::cin >> strings[i];  // 直接读取到 vector 中
    }
    
    // 输出验证
    std::cout << "读取了 " << n << " 个字符串：" << std::endl;
    for (int i = 0; i < n; i++) {
        std::cout << "第 " << i+1 << " 行: " << strings[i] << std::endl;
    }
    
    return 0;
}
```

# 二、while 循环语句

## 2.1 while 循环的基本语法

`while` 循环的基本语法结构：

```cpp
while (条件表达式) {
    // 循环体
}
```

**条件表达式**：如果条件为 `true`，则执行循环体；如果为 `false`，则终止循环。

**示例：用** **`while`** **循环计算 1 到 10 之间整数的和。**

```cpp
#include <iostream>

int main() 
{
    int i = 1;
    int sum = 0;

    while (i <= 10) {
        sum += i;
        i++;
    }

    std::cout << "1 到 10 之间整数的和: " << sum << std::endl;
    return 0;
}
```

**注意：**

- **初始化：** `while` 循环没有内置的初始化，必须在循环开始前手动初始化循环控制变量。
- **更新：** 必须在循环体内包含更新循环控制变量的语句，否则会陷入死循环
