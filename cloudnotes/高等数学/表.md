

### 1. 反正弦函数

$\frac{d}{dx} \arcsin x = \frac{1}{\sqrt{1 - x^2}}, \quad -1 < x < 1.$

### 2. 反余弦函数

$\frac{d}{dx} \arccos x = -\frac{1}{\sqrt{1 - x^2}}, \quad -1 < x < 1.$

### 3. 反正切函数

$\frac{d}{dx} \arctan x = \frac{1}{1 + x^2}, \quad x \in \mathbb{R}.$

### 4. 反余切函数

$\frac{d}{dx} \operatorname{arccot} x = -\frac{1}{1 + x^2}, \quad x \in \mathbb{R}.$

### 5. 反正割函数

$\frac{d}{dx} \operatorname{arcsec} x = \frac{1}{|x| \sqrt{x^2 - 1}}, \quad |x| > 1.$

### 6. 反余割函数

$\frac{d}{dx} \operatorname{arccsc} x = -\frac{1}{|x| \sqrt{x^2 - 1}}, \quad |x| > 1.$


### 记忆技巧

- 反正弦和反正切的导数**为正**；

- 反余弦、反余切、反余割的导数**为负**（与对应的正函数相差一个负号）；

- 分母形式与对应的正函数的导数分母形式有关（如 $\arcsin$ 对应 $\sin$ 的导数 $\cos$，但这里形式是 $\sqrt{1-x^2}$，其实是 $\cos(\arcsin x)$ 的结果）。

反三角函数的积分通常有两类：

  

1. **基本积分公式**（直接作为被积函数的结果）；

2. **通过分部积分或换元得到的含反三角函数的积分**（更常见）。

  

---

  

## 1. 基本积分（直接结果）

  

$\int \arcsin x \, dx = x \arcsin x + \sqrt{1 - x^2} + C$

$\int \arccos x \, dx = x \arccos x - \sqrt{1 - x^2} + C$

$\int \arctan x \, dx = x \arctan x - \frac12 \ln(1 + x^2) + C$

$\int \operatorname{arccot} x \, dx = x \operatorname{arccot} x + \frac12 \ln(1 + x^2) + C$

$\int \operatorname{arcsec} x \, dx = x \operatorname{arcsec} x - \ln\left(x + \sqrt{x^2 - 1}\right) + C$

$\int \operatorname{arccsc} x \, dx = x \operatorname{arccsc} x + \ln\left(x + \sqrt{x^2 - 1}\right) + C$

  

---

  

## 2. 常见变形积分

  

更常见的是形如：

$\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{a} + C$

$\int \frac{dx}{a^2 + x^2} = \frac1a \arctan \frac{x}{a} + C$

$\int \frac{dx}{x\sqrt{x^2 - a^2}} = \frac1a \operatorname{arcsec} \frac{|x|}{a} + C$

  

---

  

## 3. 推导举例

  

### 例：$\int \arcsin x \, dx$

  

用分部积分：

设 $u = \arcsin x$，$dv = dx$，则 $du = \frac{dx}{\sqrt{1-x^2}}$，$v = x$。

  

$\int \arcsin x \, dx = x \arcsin x - \int \frac{x}{\sqrt{1-x^2}} dx.$

对第二项，令 $t = 1-x^2$，$dt = -2x dx$，得

$\int \frac{x}{\sqrt{1-x^2}} dx = -\sqrt{1-x^2} + C.$

所以

$\int \arcsin x \, dx = x \arcsin x + \sqrt{1-x^2} + C.$
