
全微分是微积分里的一个重要概念，简单来说，它描述了一个**多元函数**当所有自变量都发生微小变化时，函数值的总变化量。

---

### 1. 从一元函数微分说起（类比）

-   **一元函数** $y = f(x)$ 的微分 $dy$ 是：  
    $dy = f'(x) \, dx$
    它表示：当 $x$ 变化了一点点（$dx$），$y$ 会变化多少（$dy$）。

-   **二元函数** $z = f(x, y)$ 有两个变量在变。它的**全微分** $dz$ 就是这两个方向变化量的叠加：  
    $dz = \frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dy$
    这个公式的意思是：函数的总变化 $dz$ = （由 $x$ 变化引起的部分）+ （由 $y$ 变化引起的部分）。

---

### 2. 严格的数学定义

如果函数 $z = f(x, y)$ 在点 $(x, y)$ 可微，那么它的**全微分** $df$ 定义为：
$df = \frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dy$

这里：

-   $\frac{\partial f}{\partial x}$ 和 $\frac{\partial f}{\partial y}$ 是偏导数。
-   $dx$ 和 $dy$ 是自变量的微小增量（称为微分）。

---

### 3. 物理/几何直观

想象你在爬山，山的高度是 $f(x, y)$。

-   **偏导数 $\frac{\partial f}{\partial x}$** 告诉你：如果只往东走（$x$ 方向），坡度有多陡。
-   **偏导数 $\frac{\partial f}{\partial y}$** 告诉你：如果只往北走（$y$ 方向），坡度有多陡。

**全微分 $df$** 告诉你的则是：如果你**既往东走了一小步（$dx$），又往北走了一小步（$dy$）**，你总共上升或下降了多少高度。

---


### 4. 全微分方程

**全微分方程**（也称为恰当微分方程）是指一个一阶微分方程可以写成某个函数的全微分的形式。

比如，我们有一个函数 $u(x, y) = C$（常数），对它求全微分：
$du = \frac{\partial u}{\partial x} dx + \frac{\partial u}{\partial y} dy = 0$
如果我们从方程的角度看，给了我们：
$M(x, y) dx + N(x, y) dy = 0$
如果存在一个函数 $u(x, y)$，使得：
$\frac{\partial u}{\partial x} = M, \quad \frac{\partial u}{\partial y} = N$
那么原方程就是：
$du = 0$
这意味着 $u(x, y) = C$ 就是原方程的通解（隐式解）。

这种微分方程就叫**全微分方程**。


方程：
$$M(x, y) dx + N(x, y) dy = 0$$
是全微分方程（恰当方程）**当且仅当**：
$$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$$
这个条件是判断恰当性的充要条件（假设 $M, N$ 在单连通区域内一阶连续可偏导）。


#### 例 1：恰当方程
$(2xy + 1) dx + (x^2 + 2y) dy = 0$
这里 $M = 2xy + 1$，$N = x^2 + 2y$。
$\frac{\partial M}{\partial y} = 2x, \quad \frac{\partial N}{\partial x} = 2x$
相等，所以是恰当方程。

那么找 $u$：
$\frac{\partial u}{\partial x} = 2xy + 1 \implies u = x^2y + x + h(y)$
$\frac{\partial u}{\partial y} = x^2 + h'(y) \overset{\text{应等于}}{=} x^2 + 2y \implies h'(y) = 2y \implies h(y) = y^2 + C_1$
所以：
$u = x^2y + x + y^2$
原方程等价于：
$du = 0 \implies x^2y + x + y^2 = C$
这就是隐式解。

---

#### 例 2：非恰当方程
$y dx + x dy = 0$
这里 $M = y$，$N = x$。
$\frac{\partial M}{\partial y} = 1, \quad \frac{\partial N}{\partial x} = 1$
其实这是恰当方程，因为偏导相等。

但是再看另一个：
$y dx - x dy = 0$
$M = y$，$N = -x$
$\frac{\partial M}{\partial y} = 1, \quad \frac{\partial N}{\partial x} = -1$
不相等，所以不是全微分方程。它需要乘以一个积分因子（比如 $1/y^2$ 或 $1/x^2$ 等）后才能变成恰当方程。

---

## 4. 为什么叫“全微分”方程？

因为一旦它是恰当形式：
$M dx + N dy = du$
就相当于：
$\frac{du}{ds} = 0 \ \text{沿任意路径}$
所以 $u$ 在 $xy$ 平面上的值沿着积分曲线是常数。相当于直接找到了势函数 $u$，求解就变成简单积分。










### 4. 全微分方程

如果一个方程能写成：
$M(x, y) dx + N(x, y) dy = 0$
并且它恰好是**某个函数 $u(x, y)$ 的全微分**，即：
$du = \frac{\partial u}{\partial x} dx + \frac{\partial u}{\partial y} dy = M dx + N dy$
那么原方程就变成了：
$du = 0$
这意味着函数 $u(x, y)$ 压根没变，是一个常数（$u(x, y) = C$）。



这就是为什么解全微分方程本质上是在**反推**这个函数 $u$ 是什么。

### 总结
**全微分**就是用来衡量一个多元函数，在考虑所有变量同时变化时，总共改变了多少的工具。
