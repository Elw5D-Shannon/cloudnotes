# 1.1 半导体的特性

半导体的主要材料是硅Si，锗Ge，砷化镓以及一些硫化物。
半导体内是共价键![[Pasted image 20260311205111.png]]
### 1.1.1 本征半导体

纯净，不含杂质
导电性差，温度越高导电性越高（更多自由电子)
在绝对零度下不导电
不直接用于半导体器件
**空穴：共价键中电子的空位，带正电**

## 1.1.2 杂质半导体

本征半导体中参杂质
1. N型半导体（Negative）4价半导体中掺入5价杂质元素（施主杂质）
   自由电子浓度提高，变为多数载流子（简称多子）
   N型半导体主要依靠电子导电。
   空穴称为少数载流子（少子）

2. P型半导体（Positive）4价半导体中掺入3价杂质元素（受主杂质/受主元素）
   空穴浓度提高，变为多子；电子变为少子。
   N型半导体主要依靠空穴导电。
![[Pasted image 20260311205054.png]]
在杂质半导体中，多数载流子的浓度取决于掺入的杂质浓度，而少数载流子的浓度主要取决与温度。
杂质半导体总体看仍旧保持电中性。由于外来杂质的掺入，杂质半导体的导电性能大大改善。

# 1.2 半导体二极管

## 1.2.1 PN节及其单向导电性

PN节：
    当我们将N型半导体和P型半导体结合在一起时，电子和空穴均需从浓度高的区域向浓度低的区域扩散，即N型半导体中的自由电子往P型半导体中跑，P型半导体中的空穴往N型半导体中跑，从而使得原交界面处形成了一个空间电荷区（PN结）。
    其中靠近N型半导体的N区带 + 电，靠近P型半导体的P区带 - 电，可见形成了内电场，用于阻止载流子扩散。而内电场促使少子漂移，阻止多子扩散，最后,多子的扩散和少子的漂移达到动态平衡，形成平衡PN结。在空间电荷区，由于缺少多子，所以也称耗尽层。
![[Pasted image 20260311204700.png]]

当外加电压使PN结中P区的电位高于N区的电位，称为加 **正向电压** ，简称正偏；反之称为加 **反向电压** ，简称反偏。

原理：空间电荷区的电场限制自由电子和空穴的中和，而外加电场对空间电荷区的电场产生影响。
![[Pasted image 20260311213916.png]]
（1） **PN结加正向电压时** ：空间电荷区的电场被削弱，有利于多数载流子的扩散运动，产生大的正向扩散电流。此时二极管为导通状态，**电流从P区流向N区**。

（2） **PN结加反向电压时** ：空间电荷区的电场被加强，有利于少数载流子的漂移运动。但由于少子的浓度很低，反向漂移电流也很小，此时二极管为截止状态。


PN结具有 **单向导电性** 。

## 1.2.2 二极管的伏安特性

![[Pasted image 20260311213930.png]]
阳极是Negative；阴极是Positive，一般情况下电流（正电荷）从P至N，也就是**与三角形的指向相反**


![[Pasted image 20260311231537.png|478]]
几个关键点：
1. 正向导通电压：二极管工作时，阴极电势大于阳极电势（阴极连电源正极）。
   当二极管两端电势差 V>0 且超过**正向导通电压**（硅管0.5V，锗管约0.1V）后，电流 I 随 V 指数增长，电阻变小。**此时二极管导通，两端电压视为固定，Si 0.7V，Ge 0.3V**。
   在二极管导通前，正向电流很小。
2. 正向电流由多子的扩散产生，外加的电场减弱了内电场对多子扩散的阻碍作用。Vice Versa.反向电流由少子形成，因此受到温度的影响。
3. 反向击穿电压：当二极管两端电势差小于0时，产生很小的反向电流。当反向电压超过击穿电压时，反向电流急剧增加，二极管被击穿，二极管此时**不再具有单向导电性**
4. 二极管的主要参数：最大整流电流（工作时最大正向平均电流）；最高工作反向电压（一般是击穿电压的一半）；最高工作频率（字面意思）。

## 1.2.4 稳压管

当二极管两端的反向电压超过击穿电压时（位于反向击穿区），反向电压增加很小的量便导致反向电流增加明显。

# 1.3 双极型二极管
## 1.3.1 三极管结构

三个区对应三个电极：发射区对应发射极；基区对应基极；集电区对应集电极。
用符号表示：C对应集电极，B对应基极，E对应发射极
![[Pasted image 20260314095713.png|460]]
- **发射区（对应E极）**：
    - **特点**：掺杂浓度**最高**。
    - **作用**：负责向基区发射载流子（电子或空穴）。
        
- **基区（对应B极）**：
    - **特点**：掺杂浓度**最低**，且物理厚度**极薄**（微米量级）。
    - **作用**：控制载流子的通过量，是放大作用的控制核心。
        
- **集电区（对应C极）**：
    - **特点**：掺杂浓度较高，且**面积最大**。
    - **作用**：负责收集从发射区发射过来的载流子。


![[Pasted image 20260314095853.png]]
## 1.3.1 三极管的放大作用

  由于发射区浓度极高，能向基区注入大量载流子。基区极薄，使得从发射区注入的载流子来不及在基区复合，绝大部分都能扩散到集电结边缘。集电结面积大且反偏，能有效收集扩散过来的载流子，形成集电极电流​。基极电流微小的变化，就能控制集电极电流 巨大的变化，这就是所谓的 **电流放大作用**。

所以，发射结一定正偏，集电结反偏。
