if、if-else、if-elif-else、switch等条件判断语句通过测试某个条件（通常是一个布尔表达式）并根据该条件的结果来执行不同的代码块。
# if语句

C++的if语句是条件判断语句之一，它具有多种语句形式，如if、if-else和if-else if-else，详细说明如下：

if语句：仅对某个条件进行判断，如果判断结果为false，则不执行if语句的代码块；如果判断结果为true，则执行if语句的代码块，语法格式如下：

```cpp
if (condition) {  
    // 代码块  
}
```

if-else：也是对某个条件进行判断，如果判断结果为false，则执行else语句的代码块；如果判断结果为true，则执行if语句的代码块，语法格式如下：

```cpp
if (condition) {  
    // 如果 condition 为真，执行此代码块  
} else {  
    // 如果 condition 为假，执行此代码块  
}
```


```cpp
if (condition1) {  
    // 如果 condition1 为真，执行此代码块  
} else if (condition2) {  
    // 如果 condition1 为假，但 condition2 为真，执行此代码块  
} else {  
    // 如果 condition1 和 condition2 都为假，执行此代码块  
}
```


```cpp
if (condition1) {  
    // 如果condition1为真，执行此代码块  
} else if (condition2) {  
    // 如果condition1为假，但condition2为真，执行此代码块  
} else if (condition3) {  
    // 如果condition1和condition2为假，但condition3为真，执行此代码块  
} else {  
    // 如果condition1、condition2和condition3都为假，执行此代码块  
}
```

使用if语句实现成绩划分，根据输入的成绩将其划分为不同的等级，示例代码如下：

```cpp
#include <iostream>

using namespace std;

int main() {
    int score;
    cout << "请输入成绩：";
    cin >> score;

    if (score >= 90) {
        cout << "成绩等级：优秀" << endl;
    } else if (score >= 80) {
        cout << "成绩等级：良好" << endl;
    } else if (score >= 70) {
        cout << "成绩等级：中等" << endl;
    } else if (score >= 60) {
        cout << "成绩等级：及格" << endl;
    } else {
        cout << "成绩等级：不及格" << endl;
    }
}
```


在实际应用中，if语句的条件判断经常使用逻辑运算符（如 &&（逻辑与）、||（逻辑或）和 !（逻辑非））对多个条件同时进行判断，其语法格式如下：

```cpp
if (condition1 && condition2) {  
    // 如果condition1和condition2都为true，则执行此代码块  
} else if (condition3 || condition4) {  
    // 如果condition3或condition4为true，则执行此代码块  
} else if (!condition5) {  
    // 如果condition5为false，则执行此代码块  
} else {  
    // 上述条件皆不通过，则执行此代码块  
}
```

C++的switch语句也是条件判断语句之一，它只有一种语句形式，并且switch语句必需搭配关键字case和break使用，语法格式如下：

```cpp
switch (expression) {  
    case value1:  
        // 当 expression 的值为 value1 时，执行此代码块  
        break;  
    case value2:  
        // 当 expression 的值为 value2 时，执行此代码块  
        break;  
    default:  
        // 当 expression 的值不是任何 case 标签时，执行此代码块  
        break;  
}
```

我们将成绩划分改用switch语句实现，示例代码如下：

```cpp
#include <iostream>
using namespace std;
int main() {
    int score;
    cout << "请输入成绩：";
    cin >> score;
    switch (score / 10) {
        case 10:
        case 9:
            cout << "成绩等级：优秀" << endl;
            break;
        case 8:
            cout << "成绩等级：良好" << endl;
            break;
        case 7:
            cout << "成绩等级：中等" << endl;
            break;
        case 6:
            cout << "成绩等级：及格" << endl;
            break;
        default:
            cout << "成绩等级：不及格" << endl;
            break;
    }
}
```


# 三目运算符

我们知道C++的三目运算符是一种简短的if-else语句，用于根据条件选择两个值中的一个，它的语法如下：

```cpp
condition ? value_if_true : value_if_false;
```

三目运算符也可以实现if-else嵌套功能，也就是说也能对多个条件进行判断处理，我们还是以成绩划分为例，使用三目运算符的实现过程如下：

```cpp
#include <iostream>
using namespace std;
int main() {
    int score;
    cout << "请输入成绩：";
    cin >> score;
    string value;
    value = (score >= 90) ? "成绩等级：优秀" :  (score >= 80) ?  "成绩等级：良好" : (score >= 70) ?  "成绩等级：中等" : (score >= 60) ?  "成绩等级：及格" : "成绩等级：不及格";
    cout << value << endl;
}
```

运行上述代码，在运行界面输入相应数字即可得到相应成绩等级，如图所示

从上述代码分析发现，三目运算符的if-else嵌套语法如下：

```cpp
condition ? value_if_true : condition1 ? value_if_true1 : condition2 ? value_if_true2 : value_if_false2;
```
