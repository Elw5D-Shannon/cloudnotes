
我来帮你编写几种不同语言的代码来遍历逐行输入的带有空格的整数：


## Python 版本

  
```python

# 方法1：读取直到EOF

import sys

  

print("请输入整数（每行可以包含多个空格分隔的整数，按Ctrl+D或Ctrl+Z结束）：")

for line in sys.stdin:
    # 去除首尾空格，按空格分割，转换为整数
    numbers = [int(x) for x in line.strip().split()]
    if numbers:  # 如果该行有数字
        print(f"该行数字: {numbers}")
        # 遍历每个数字
        for num in numbers:
            print(f"处理数字: {num}")

  

# 方法2：指定行数

def read_n_lines(n):
    print(f"请输入{n}行整数（每行可以包含多个空格分隔的整数）：")
    for i in range(n):
        line = input(f"第{i+1}行: ").strip()
        numbers = list(map(int, line.split()))
        print(f"第{i+1}行数字: {numbers}")
        for num in numbers:
            print(f"  处理: {num}")

# 使用示例
# read_n_lines(3)

```

  

## C++ 版本

  

```cpp

#include <iostream>
#include <sstream>
#include <string>
#include <vector>
using namespace std;

int main() {
    string line;
    vector<int> numbers;
    cout << "请输入整数（每行可以包含多个空格分隔的整数，按Ctrl+Z结束）：" << endl;
    // 读取每一行
    while (getline(cin, line)) {
        if (line.empty()) continue;
        // 使用字符串流解析当前行
        istringstream iss(line);
        int num;
        numbers.clear();
        cout << "当前行数字: ";
        while (iss >> num) {
            numbers.push_back(num);
            cout << num << " ";
        }
        cout << endl;
        // 遍历当前行的数字
        for (int n : numbers) {
            cout << "处理数字: " << n << endl;
        }
    }
    return 0;
}
```

#### 编程要求

本关的编程任务是完成文件 step6/upvote.cpp 的编写，以实现朋友圈点赞的功能。 upvote.cpp 文件为空，需要你从头开始编写。具体要求如下：

- **程序的输入格式为：**

第一行输入一个正整数 N （ N < 1000 ），是该用户点赞的文章数量。

随后的 N 行，每行给出一篇被其点赞的文章的标签描述，格式为`“k S1​ S2​ ......Sk​”`。

`k`表示该文章的标签数，`1≤k≤10`，即每篇文章最多10个标签； `Sk​`是标签，是1到1000之间的整数，即总共有不超过1000种标签。

所有数值之间以空格分隔。

- **程序的输出格式为：**

统计所有被点赞的文章中最常出现的那个标签，在一行中输出它的**标签编号**和**出现次数**，数字间隔1个空格。如果有并列，则输出编号最大的那个。

例如：

1. `4`
2. `3 889 233 2`
3. `5 100 3 233 2 73`
4. `4 3 73 889 2`
5. `2 233 123`

输出结果为：`233 3`

上述示例中出现最多次（3次）的标签有233和2，存在并列时，输出最大的那个标签233，然后输出其出现的次数3。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef struct
{
    int name;
    int times;
}label;
vector<label> labels;
int main(){
    int num;
    cin >> num;
    string line;
    // 读取第一行后面的换行符
    getline(cin, line); // 消耗掉第一行末尾的换行符
    // 读取接下来的N行
    for(int article = 0; article < num; article++) {
        getline(cin, line);
        if (line.empty()) {
            article--; // 如果是空行，重新读取
            continue;
        }

        istringstream iss(line);
        int count;
        iss >> count; // 读取该行的标签数量
        int tem;
        // 读取该行的所有标签

        for(int i = 0; i < count; i++) {
            iss >> tem;
            // 查找是否已存在该标签
            bool found = false;
            for (int j = 0; j < labels.size(); j++) {
                if (labels[j].name == tem) {
                    labels[j].times++;
                    found = true;
                    break;
                }
            }

            // 如果是新标签，添加到vector
            if (!found) {
                label newLabel;
                newLabel.name = tem;
                newLabel.times = 1;
                labels.push_back(newLabel);
            }
        }
    }

    // 查找出现次数最多的标签

    int maxTimes = -1;
    int maxLabel = -1;
    for (int i = 0; i < labels.size(); i++) {
        if (labels[i].times > maxTimes) {
            maxTimes = labels[i].times;
            maxLabel = labels[i].name;
        }
        else if (labels[i].times == maxTimes) {
            // 如果出现次数相同，选择编号更大的
            if (labels[i].name > maxLabel) {
                maxLabel = labels[i].name;
            }
        }
    }
    // 输出结果
    cout << maxLabel << " " << maxTimes << endl;
    return 0;
}
```

## map 是什么？

  

`map` 是 C++ 标准模板库（STL）中的一种**关联容器**，它存储的是**键值对**（key-value pair）的数据结构。

  

## 形象理解

  

可以把 `map` 想象成一个**字典**或**电话本**：

- **键（key）**：就像人的名字

- **值（value）**：就像对应的电话号码

  

```cpp

// 电话本示例

map<string, string> phonebook;

phonebook["张三"] = "13800138000";

phonebook["李四"] = "13900139000";

  

// 查找张三的电话

cout << phonebook["张三"]; // 输出：13800138000

```

  

## 在这个题目中的应用

  

```cpp

map<int, int> count; // 标签 -> 出现次数

```

  

这行代码创建了一个 `map`，其中：

- **键（key）**：标签编号（int类型）

- **值（value）**：该标签出现的次数（int类型）

  

### 具体工作过程：

  

```cpp

// 假设输入：3 889 233 2

// 表示这篇文章有3个标签：889、233、2

  

count[889]++; // 标签889出现次数+1

count[233]++; // 标签233出现次数+1  

count[2]++;   // 标签2出现次数+1

```

  

### 执行过程示例：

  

```cpp

#include <bits/stdc++.h>

using namespace std;

  

int main() {

    map<int, int> count;

    // 处理第一篇文章：3 889 233 2

    count[889]++; // count变成：{889:1}

    count[233]++; // count变成：{889:1, 233:1}

    count[2]++;   // count变成：{889:1, 233:1, 2:1}

    // 处理第二篇文章：5 100 3 233 2 73

    count[100]++; // {889:1, 233:1, 2:1, 100:1}

    count[3]++;   // {889:1, 233:1, 2:1, 100:1, 3:1}

    count[233]++; // 233变成2次：{889:1, 233:2, 2:1, 100:1, 3:1}

    count[2]++;   // 2变成2次：{889:1, 233:2, 2:2, 100:1, 3:1}

    count[73]++;  // {889:1, 233:2, 2:2, 100:1, 3:1, 73:1}

    // 最终count的内容：

    // 889:1次

    // 233:2次

    // 2:2次

    // 100:1次

    // 3:1次

    // 73:1次

    return 0;

}

```

  

## map 的特点

  

### 1. **自动排序**

```cpp

map<int, int> m;

m[5] = 1;

m[2] = 1;

m[8] = 1;

// map会自动按键排序：{2:1, 5:1, 8:1}

```

  

### 2. **键唯一**

```cpp

map<int, int> m;

m[233] = 1; // 第一次插入

m[233] = 2; // 会覆盖之前的值，变成{233:2}

```

  

### 3. **直接访问**

```cpp

map<int, int> m;

m[888]++;    // 如果888不存在，会自动创建并初始化为0，然后+1变成1

m[888]++;    // 888存在，直接+1变成2

```

  

### 4. **遍历map**

```cpp

// 方法1：范围for循环

for(auto& pair : count) {

    cout << "标签：" << pair.first;    // 键（标签编号）

    cout << "次数：" << pair.second;   // 值（出现次数）

}

  

// 方法2：使用迭代器

for(auto it = count.begin(); it != count.end(); it++) {

    cout << "标签：" << it->first;

    cout << "次数：" << it->second;

}

```

  

## 为什么用 map 比用 vector 好？

  

### 用 vector 的方式：

```cpp

vector<label> labels;

// 每次都要遍历查找

for (int i = 0; i < labels.size(); i++) {

    if (labels[i].name == tem) {

        // 找到标签

    }

}

// 时间复杂度：O(n)

```

  

### 用 map 的方式：

```cpp

map<int, int> count;

count[tem]++; // 直接访问，不需要查找

// 时间复杂度：O(log n)

```

  

**简单来说**：`map` 让你可以直接通过标签编号找到它的出现次数，不需要遍历整个容器去查找，效率更高，代码也更简洁。
